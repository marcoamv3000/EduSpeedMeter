SpeedMeter
==========

Descrição
---
Este é o repositório para o aplicativo Android EduSpeedMeter.
Pode exibir algumas informações úteis sobre sua rota:
* A velocidade real, em tempo real;
* O tempo decorrido;
* Quantos metros percorreu;
* Sua velocidade máxima;
